# Receipts

Surfboard offers an efficient receipt management system that provides users with easy access to detailed transaction information. With the [**Receipts APIs**](/api/receipts?lang=cURL), you can efficiently track and manage both digital and physical receipts, which are essential for returns and refunds.

# Key features

Our receipts APIs offers key features as follows:

-   **Print receipts**
-   **Email receipts**
-   **Receipts through QR code**
-   **Multi language support**
-   **Customization of receipt information**

## Email receipt

You can use [**Email receipt API**](/api/receipts?lang=cURL#Email-Receipt) to send digital receipts to customers via email after a transaction or a purchase.

## Print receipt

Our in-store payment solution provides a [**Print Receipts API**](/api/receipts#Print-Receipt) that enables direct printing of receipts and invoices from your POS system, either through printing dock attached to SurfTouch or through SurfPrint, our payment terminal with printer attached to it.

## Add receipt information

With [**Add Receipt Information API**](/api/receipts#Add-Receipt-Information) you can add cash register details to the receipt. This receipt information typically includes the following:

-   **Sequence number**-sequence number of the receipt
-   **Cash register name**- designated name of cash register
-   **Control unit serial number**-control unit manufacturing number

## Fetch receipt link

The [**Fetch Receipt Link API**](/api/receipts) allows you to retrieve the URL of the digital receipt for a transaction using the transaction, payment, or order ID.
