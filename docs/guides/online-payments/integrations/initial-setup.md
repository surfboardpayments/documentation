# Getting Started with Surfboard Payments

This guide will help you get started with Surfboard Payments integration by walking you through the initial setup process.

## Initial Setup

### 1. Create a Developer Account

1. Click on **Sign Up** to create a new account and fill in the required details.

> If you have already created an account **login** to the developer portal

### 2. Generate API Credentials

Once logged in to the Developer Portal:

1. Navigate to the **Console** section
2. Select **API keys** from the sidebar
3. Click **Create New Key** in the demo environment
4. Add a name of your desire to identify the keys you generate and copy the API key and API secret and store it securely.

{% callout type="caution" label="Security Important" %}
Never share your API credentials or commit them to version control. Store them securely using environment variables or a secure key management system.
{% /callout %}

Once you generate the API credentials, you can proceed further implementation and test our APIs in the demo environment.
