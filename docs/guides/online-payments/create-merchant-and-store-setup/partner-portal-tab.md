This is similar to our API method whereas you can generate the merchant application using the Partner portal.

Partners must generate a unique onboarding link for each merchant through the Partner Portal. This link allows merchants to access their application and start the onboarding process.

To generate an onboarding link

1. Log in to your **Partner Portal** account.
2. Navigate to the **Applications** section.
3. Click **Create Application**, select the checkbox to enable online payments for your store at the bottom and fill in the mandatory fields, and select Create Application.
4. Share the KYB link with your merchant to complete their onboarding process.

> The link remains valid for 30 days.

{% callout type="note" label="Note" %}
You also have the flexibility to customise the merchant onboarding page to reflect your brand identity.
{% /callout %}
