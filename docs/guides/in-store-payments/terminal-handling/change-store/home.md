Once terminal is successfully registered under a merchant, it cannot be reassigned to a different merchant, you can move it between different stores belonging to the same merchant. This is useful if a terminal needs to be relocated to a new location. This can be done using

- Using API
- Using Partner Portal
- Using Merchant Portal

> Change store can happen only within a merchant.
