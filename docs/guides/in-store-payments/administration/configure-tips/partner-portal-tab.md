**For Merchant-Level Configuration**

1. Merchant-level settings are configured by navigating to **Merchants→ select the desired merchant→ Configurations,** .
2. Goto **Set Merchant Tips Config** fill in the details, and click **Save Changes** to apply.

**For Terminal-Level Configuration**

1. Configure terminal-level settings by navigating to **Merchants→ select the desired merchant→Terminals**.
2.  In Terminals section **Actions → Set Terminal Tips Config** for the terminal you want to configure tips, fill in the details, and click **Save Changes** to apply.

**For Store-Level Configuration**

1. Set store-level configuration can be done by clicking **Merchants → select the desired merchant→** **Stores**.
2. Select the desired store in **Store** section, click **Actions** for the desired terminal to configure tips.
3. Provide the needed details for setting in the store-level tip configs and click **Save Changes**.
