An order is created against the **`terminalId`** from terminal registration and includes line items with prices. Each payment attempt must reference the **`orderId`**, making the order the foundation of the payment flow.

### Prerequisites

- API Credentials and  **`merchantId`** are required for API access.
- **`terminal$id`**: Id of the registered terminal obtained via [**Register Terminal API**](https://developers.surfboardpayments.com/api/terminals?lang=cURL#Register-Terminal).
