# Merchant Functions

Merchant functions are the essential tools to manage and configure merchant-related settings with surfboard payments, so you can manage the various functionalities available for merchant management. They are

- Setup Multi-merchant group
- Update Merchant details
- Activate Terminal
- Additional merchant functions
