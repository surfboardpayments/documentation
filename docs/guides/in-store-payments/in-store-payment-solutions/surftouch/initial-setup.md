1. Once the terminal is on, tap **Select a Wi-Fi network**. Choose your preferred network from the list to establish an internet connection.
    
   {% callout type="tip" label="Tip" %}
   SurfTouch is equipped with an inbuilt SIM card and can use mobile data if Wi-Fi is unavailable.
   {% /callout %}
    
2. After connecting to the server, tap **Register Terminal** when prompted on the screen.
3. The terminal displays a **6-digit OTP**. The merchant must use this code to complete the terminal registration process through the Partner Portal.
4.  Following successful registration, the SurfTouch automatically downloads and applies necessary updates and configurations.
5.  The device will then automatically transition into its assigned operating mode: **CheckoutX** (integrated) or **SurfPay** (standalone).

{% callout type="note" label="Note" %}
In the standalone mode, SurfTouch takes care of the entire process from order creation to payment completion.
{% /callout %}
