This section describes the various configurations and customization for your SurfPad.

| Functionality | Configuration Details |
| --- | --- |
| **Display Logo** | Logo must be **320x240px** and in **GIF format**. Email your logo to the integration address for platform update. |
| **Status Bar & Colors** | **Visibility**: Set `showStatusBar` parameter (`TRUE`/`FALSE`). 
**Color Schemes**: Use Hex codes for **`statusBarBackground`**, **`statusBarForeground`** (text/icons), and **`statusBarBatteryLow`** (for battery icon below 20%). |
| **Auto Sleep** | Sets inactivity timeout while on battery to conserve power. Configurable via the Merchant Dashboard. Press any key to wake. |
| **Shutdown Time** | Default auto-restart is at **2 AM** (device timezone). Device must restart at least once every 24 hours. Manual shutdown via long-press of the **Power key**. |
| **Configuration Updates** | Successful over-the-air updates are signaled by the **indicator LEDs flashing in a wave pattern**. |
| **SurfPad Info Screen** | Access by pressing **1 3 3 7** from the idle screen. Displays technical details like battery, network status, and **Terminal ID**. Press **Cancel** to exit. |
