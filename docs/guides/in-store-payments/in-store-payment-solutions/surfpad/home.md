The SurfPad is a handheld payment acceptance terminal from Surfboard Payments featuring an integrated smart and contactless card reader.

SurfPad supports:

- Card payments
- Contactless payments
- API integration with your existing systems (ECR, checkout systems etc.)
- Integration with our pre-built apps (CheckoutX and CheckoutPro)
- Connection with WiFi or mobile network

This section outlines the initial setup of SurfPad and includes important usage instructions.

{% callout type="caution" label="Security Important" %}
Any unauthorized alteration, even minor physical changes (e.g., a loose screw), will trigger security mechanisms, resulting in the terminal being **flagged as compromised and permanently blocked** from use.
{% /callout %}
