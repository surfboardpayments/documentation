This section shows how to identify various errors on the SurfPad and provides instructions to resolve them

| Error Type | Cause | Troubleshooting/Resolution |
| --- | --- | --- |
| **Connection Errors** | Issues with Wi-Fi/mobile network or hardware connectivity. | Check your network connection. If the problem persists, **restart the terminal** and try again, potentially forcing a connection retry using the key sequences (Section: Setup). |
| **Card Read Errors** | Card is dirty, damaged, or improperly inserted/tapped. | Ensure the card is clean and undamaged. For contactless issues, wait a few seconds before retrying the tap. |
| **Payment Errors** | Insufficient funds, network instability, or payment gateway issues. | Resolve the underlying issue (e.g., check funds, verify network stability) and **retry the payment**. |
