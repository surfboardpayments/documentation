# Change Store

Once terminal is successfully registered under a merchant, it cannot be reassigned to a different merchant, you can move it between different stores belonging to the same merchant. This is useful if a terminal needs to be relocated to a new location.

> Change store can happen only within a merchant.

#### Using API

Changes the store to which the terminal is registered. You can use this API to utilize the terminal in another store created under the same merchant. However, the Terminal ID remains unchanged.

#### Using Partner Portal

1. Login to the partner portal, goto **Merchants**→select the desired merchant→**Terminals** in the same page.
2. Select the terminal you want to move to another store, then click **Change Store**.
3. In the side tray on the right, enter the **Store ID** where the terminal should be registered and click **Update**.

#### Using Merchant Portal

1. Login to the **Merchant Portal**, goto **Terminals**→select the terminal you want to move to another store.
2. Goto **Settings**→ **Change store** where the terminal should be registered.
3. Select the desired store from the dropdown list and click **Update Terminal**.
