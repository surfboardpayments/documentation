### Guide

# Update Terminal Name

Use the Update Terminal Name API to change the registered terminal's name specifying the new terminal name. You can also update the name

#### Using API

`PUT` request by providing the `terminalId` and the new name for the terminal you want to update.

#### Using Partner Portal

1. Login to the **Partner Portal**, goto **Terminal Info** tab.
2. Enter the terminalId or serial number of the terminal to be named, click **Get Terminal Details**.
3. Click the edit icon. In the pop-up sheet on the right, enter the **Terminal Name** and click **Update Name**.

#### Using Merchant Portal

1. Log in to the **Merchant Portal**, go to **Terminals**→select the terminal you want to rename.
2. Click the edit icon in the top opens a pop up, enter the new name, and click **Update Name**.
