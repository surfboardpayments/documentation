
### Description

Setting tips configurations for a specific store.

### Specifications

### Request Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `tipConfig` | string | No | Specifies how tips are managed during the payment. The possible values are the following: |
| `tipLevel1` | number | No | Specifies the first preset percentage value for tips. The values should be set as whole numbers representing percentages, e.g., 10 for 10%. This number represents a percentage of the total amount that can be chosen by the customer as a tip. It will be the first option displayed to customers. |
| `tipLevel2` | number | No | Specifies the second preset percentage value for tips. The values should be set as whole numbers representing percentages, e.g., 20 for 20%. This number represents a percentage of the total amount that can be chosen by the customer as a tip. It will be the second option displayed to customers. |
| `tipLevel3` | number | No | Specifies the third preset percentage value for tips. The values should be set as whole numbers representing percentages, e.g., 30 for 30%. This number represents a percentage of the total amount that can be chosen by the customer as a tip. It will be the third option displayed to customers. |
| `defaultCustomAmount` | number | No | A custom amount that will be prefilled on the screen for your ease. This is only available when freeAmountEnabled is set to true and tips enabled. |
| `displayCalculatedAmount` | string | No | Specifies whether or not to display calculated tip amount on screen. |
| `tipDisplayFormat` | string | No | The format in which the tip is displayed. |
| `freeAmountEnabled` | boolean | No | Set this to true to enable the option for entering a custom amount on the tips screen. |

# Store Tips Config

Set Store Tips Config allows you to customize tip settings for a specific store. By configuring store-level tips, you can ensure consistent tip application across all terminals within that store.

#### Using API

Sets tips configurations for all the terminals registered to a specific store. Use this API to configure the same tips configurations across multiple terminals associated with a specific store.

#### Using Partner Portal

1. Set store-level configuration can be done by clicking **Merchants → select the desired merchant→** **Stores**.
2. Select the desired store in **Store** section, click **Actions** for the desired terminal to configure tips.
3. Provide the needed details for setting in the store-level tip configs and click **Save Changes**.
