# Deactivate Payment Method

The Deactivate Payment Method endpoint is used to disable one or more active payment methods for a merchant. It allows you to turn off specific payment methods in a single request, ensuring that only the required payment options remain available for transactions.

#### Using API

Create a `DELETE` request to **Deactivate Payment Method API** to deactivate the payment method added to the merchant or store using its Payment Method ID.

#### Using Partner Portal

1. Log in to the **Partner Portal** → **Merchants** → select the desired merchant, then navigate to **Payment Methods**
2. In the page you choose the payment method, which you want to delete and click the **Delete** icon at the right corner.
