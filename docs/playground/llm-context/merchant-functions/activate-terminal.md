### Guide

# Activate Terminal

This API is available for partners who manage their own storage and terminal delivery solution and requires a PCI compliant storage setup. The API can be used to activate terminals that are tagged to you as a partner. This API will return an error if the entered serial number is not available to you. You can activate your terminal 

#### Using API

Post request with the `serialNo` of the terminal which is to be activated

#### Using Partner Portal

1. Log in to the **Partner Portal**, navigate to **Merchants** tab and select the desired merchant.
2. In the merchant's page, goto **Terminals** section, click **Activate Terminal**.
3. Opens a side tray, now enter the serial number of the terminal, to be activated. Here you activate one or more terminals simultaneously.
