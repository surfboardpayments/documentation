# Update Merchant Details

For merchants who wish to make changes to their merchant information, can update certain basic information .

#### Using API

To update merchant details, You can use this API to edit the email address, merchant logo URL, and phone number of a merchant.

#### Using Partner Portal

1. Log in to the **Partner Portal**, navigate to **Merchants** tab and select the desired merchant.
2. In the merchant's page, below the **Merchant Info** section, click **Edit** icon.
3. Opens a side tray, now edit your merchant information.
