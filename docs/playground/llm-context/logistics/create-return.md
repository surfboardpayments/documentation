# Device Returning

If case of receiving a malfunctioning or defective terminal, you can return it to Surfboard and track its status using either the Partner Portal or API.

## Create Return

#### Using Create Return API

Creates a return request for terminals using the API. This enhanced version allows for more detailed merchant information and multiple return order lines.

#### Using Partner Portal

1. Log into the **Partner Portal**, go to **Terminal Info**, search for the terminal by its ID(obtained from terminal registration) or serial number (located on the back of the hardware terminal) and click **Get Terminal Details** to view terminal info.
2. Select **Return Order** in the top-right corner and fill in the required details and click **Return Order**.

You can check the return status using the Get Return Status API.
