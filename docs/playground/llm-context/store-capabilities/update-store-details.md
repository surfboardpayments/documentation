# Update Store Details

After a store is created, you can update specific details to keep your information current. Update store details

#### Using API

Send a `PUT` request to the Update Store API, providing the storeId and the updated details in the request body

#### Using Partner Portal

1. Login **Partner Portal**, go to **Merchants** → select the desired merchant → **Stores**
2. Click the **Edit** icon → Update store details in the side tray → Click 'Update Store' to save.
