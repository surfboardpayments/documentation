### Request Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `templateId` | String | No | Select from a default set of templates offered by Surfboard. |
| `terminalId` | String | No | Print receipt on any printing enabled Surfboard terminal using this param. |
| `language` | String | No | Prints the receipt in the given language. The available languages are, 'sv' , 'da' , 'fi' , 'en'. Default is the merchant language. |

# Print Receipts

Print Receipts provides a solution that enables direct printing of receipts and invoices from your POS system, either through SurfTouch, which features a dock for easy checkout, or through SurfPrint, which has a built-in printer for on-floor payments.
