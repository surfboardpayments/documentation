# NFC Reading

The NFC Reading API enables reading NFC tags linked to an order, allowing tagged items to be automatically identified and recorded during a purchase, which streamlines item capture and reduces manual scanning for NFC-enabled products.
You can create the NFC tags using

#### Create Read Session

This is used to read all NFC tags on products to identify the complete list of purchased items. If you want to read a single product, you can set the mode to single.

#### Order with read tag

An order is created by including readTags in the order creation request, the system indicates that NFC tag reading will be used to identify products for this specific order
