# Print Receipts

Print Receipts provides a solution that enables direct printing of receipts and invoices from your POS system, either through SurfTouch, which features a dock for easy checkout, or through SurfPrint, which has a built-in printer for on-floor payments.

{% callout type="note" label="Note" %}
The :id in url supports Transaction, Payment and Order ID's.
{% /callout %}
