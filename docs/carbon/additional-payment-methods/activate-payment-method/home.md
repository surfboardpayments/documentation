# Additional Payment Methods

Our in-store solution supports multiple payment methods, with card payments enabled by default for all merchants. You can use the Payment Methods API or the Partner Portal to add or remove other payment methods like AMEX and SWISH. These options can be set for specific merchants or stores to give you flexible control over how customers pay.

## Activate Payment Method

It is used to enable payment methods other than the default card option for a merchant. It allows you to activate multiple payment methods in a single request and apply specific configurations to each method as needed, giving you flexible control over the available payment options.
