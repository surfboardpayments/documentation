# Setup Multi-merchant group

A **multi-merchant terminal** allows several businesses to process payments on a single physical device. This is ideal for locations like a music concerts, where each vendor can use the same terminal, but their individual transactions are routed to their separate accounts. This approach helps reduce hardware costs and simplifies the checkout process.

To set up a multi-merchant group, follow these steps:

1. Create a multi-merchant group
2. Add merchants to the multi-merchant group
3. Register a terminal to the multi-merchant group
4. Processing payments

## 1. Create a Multi-merchant group

You can create a multi-merchant group using the Create Multi-Merchant Group API. This API creates an application that allows multiple merchants to use the same terminal in a single location.
