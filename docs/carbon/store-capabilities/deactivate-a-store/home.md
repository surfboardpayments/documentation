# Deactivate a Store

You can deactivate any store under a merchant. You can delete a store instantly or schedule it to be deleted at a later date. Stores can only be deleted if there are no terminals registered to them. If a store has active terminals, you must first delink them or move them to another store under the same merchant. Deactivate a store
