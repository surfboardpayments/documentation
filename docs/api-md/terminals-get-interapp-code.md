# Get Interapp Code

## Overview
Use this API to generate registration code used for interapp registration.

## Prerequisites
- A valid `merchantId` and `storeId` are required to identify the target store and merchant.
- API Key and Secret authentication is required.
- Ensure you have the correct `MERCHANT-ID` header set.

## Request

### HTTP Method and URL
```
GET /merchants/:merchantId/stores/:storeId/terminals/interapp
```

### Headers
| Header | Value | Required |
|---|---|---|
| Content-Type | application/json | Yes |
| API-KEY | YOUR_API_KEY | Yes |
| API-SECRET | YOUR_API_SECRET | Yes |
| MERCHANT-ID | YOUR_MERCHANT_ID | Yes |

### Query Parameters
*None*

### Request Body Parameters
*None*

### Request Example
```json
{
  "Content-Type": "application/json",
  "API-KEY": "YOUR_API_KEY",
  "API-SECRET": "YOUR_API_SECRET",
  "MERCHANT-ID": "YOUR_MERCHANT_ID"
}
```

## Response

### Response Parameters
| Parameter | Type | Description |
|---|---|---|
| status | string | Status of the request. |
| data | object | Response data. |
| data.registrationCode | string | Returns the registration code for interapp. This code is valid for 120 seconds. |
| message | string | A message that describes the status of the request. |

### Success Response Example
```json
{
	"status": "SUCCESS",
	"data": {
		"registrationCode": "v_Z37lV0JatqBvou"
	},
	"message": "Registration Code Generated Successfully"
}
```

## Error Responses

*No specific error codes provided in the JSON data.*

## Notes
- The registration code generated by this API is only valid for 120 seconds.
